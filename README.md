# Bad Faith Data

A data journalism project disguised as a conspiracy zine. Each issue investigates the gap between institutional data signals and what publicly available proxy data actually shows.

Built with React and D3.

## Issues

**01: The Cart You Left Behind**
Grocery stores as manipulation machines. What happens to the tricks when the economy breaks. Maps the architecture of retail psychology against proxy signals for economic stress that official consumer confidence numbers don't capture.

**02: Disclosure Season** *(coming soon)*

## Stack

React, Vite, D3.js, Framer Motion, Cloudflare Pages

## Development

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run build
```

`dist/` deploys to Cloudflare Pages at zine.gabyhernandez.dev.
